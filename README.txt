🔧 INSTRUCȚIUNI DEPLOY VERCEL - ASISTENT DECOIMOB

1. Intră pe https://vercel.com și loghează-te (cu GitHub sau email).
2. Creează un nou proiect și importă acest folder.
3. În Dashboard > Settings > Environment Variables, adaugă:
   - Name: OPENAI_API_KEY
   - Value: cheia ta OpenAI (începe cu sk-...)
4. Apasă „Deploy”.

Integrează în pagina ta HTML codul din `chat-widget.html`.

Backendul va fi accesibil la: https://nume-proiect.vercel.app/api/chat
